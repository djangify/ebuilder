# Open Source Policy

Djangify eCommerce Builder (aka eBuilder) is an open-source project.

You are free to:
- Self-host (with or without Adminita Django Admin)
- Modify the code
- Use it for personal or client projects

The project is funded through:
- Optional paid licenses (branding removal)
- Optional managed hosting
- Custom build setup

No functionality is restricted in the open-source version.
